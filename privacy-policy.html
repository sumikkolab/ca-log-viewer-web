取扱方針の概要

  • 本アプリは Microsoft Graph API 経由でサインインログを取得し、ローカル端末上でのみ処理します。外部サーバーへの送信は行いません。
  • UPN (User Principal Name) の平文をデータベースに保存しません (Phase 10 PII 方針)。表示は短期暗号化キャッシュ (Windows DPAPI、TTL 7 日) 経由で行います (Phase 13 確定方針)。
  • 取得したログは利用者の Windows ユーザープロファイル配下の SQLite DB に保存され、利用者の操作で全件削除可能です。
  • テスト用デモデータ (110 件 / 6 シナリオ) は *.example ドメイン + RFC 5737 IP のみを使用し、実テナント情報を含みません。Microsoft Store 配布版にはデモデータは含まれません。

Summary of data handling

  • This app fetches sign-in logs via Microsoft Graph and processes them locally only. No external transmission.
  • Plain UPNs are not stored in the database (Phase 10 PII policy). Display is via a short-term encrypted cache (Windows DPAPI, TTL 7 days, Phase 13).
  • Fetched logs are stored in a SQLite DB under the user's Windows profile and can be fully deleted by the user.
  • Test demo data (110 records / 6 scenarios) uses only *.example domains + RFC 5737 IPs, contains no real tenant information, and is excluded from Microsoft Store builds.
