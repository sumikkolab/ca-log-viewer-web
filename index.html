CA Log Viewer

Conditional Access のサインインログを現場で読める

Microsoft Entra ID のサインインログから Conditional Access 関連を抽出し、CA 評価結果・適用ポリシー・端末準拠状態を画面と HTML 監査レポートで可視化する Windows デスクトップアプリ。

Read Conditional Access sign-in logs in context

A Windows desktop app that fetches Entra ID sign-in logs via Microsoft Graph and visualizes Conditional Access related entries — CA evaluation result, applied policies, device compliance — on screen and in HTML audit reports.
